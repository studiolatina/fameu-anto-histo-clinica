## Sobre o projeto

Desenvolvimento de uma plataforma digital interativa que integra conteúdos de Anatomia, Histologia, Embriologia e Clínica de forma progressiva e estruturada, promovendo integração vertical do conhecimento morfológico na formação médica.

O projeto é construído de forma longitudinal ao longo de múltiplos semestres. Neste repositório estão registrados o processo de trabalho, a produção científica e o desenvolvimento técnico da plataforma.
